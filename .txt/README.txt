# Proyecto de Cálculo de IMC

## Descripción
Esta aplicación permite calcular el Índice de Masa Corporal (IMC) utilizando una interfaz gráfica de usuario desarrollada con Tkinter. La aplicación está diseñada para ser fácil de usar y proporciona resultados precisos basados en la información ingresada por el usuario.

## Instrucciones de Instalación
1. Asegúrese de tener Python 3.x instalado en su sistema.
2. Instale las dependencias necesarias ejecutando el siguiente comando en la línea de comandos:
   pip install pillow

## Ejecución del Programa
1. Descargue y descomprima el archivo .zip proporcionado.
2. Navegue hasta la carpeta descomprimida y ejecute el archivo `programa.exe` para iniciar la aplicación.
3. Alternativamente, puede ejecutar el programa desde el código fuente usando Python:
   python nombre_del_archivo.py

## Dependencias
- Python 3.x
- Pillow (PIL Fork)
